# IMDb Movie Recommendation System

## Project Overview
This project builds a movie recommendation system for streaming platforms based on the IMDb Top 250 dataset.

## Objective
To identify popular movies and recommend top-performing films to maximize audience engagement.

## Methodology
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Genre Analysis
- Feature Engineering
- Recommendation Model

## Key Findings
- High-rated movies (8+) perform better
- Movies with high votes attract more viewers
- Popular genres include Action, Adventure, Sci-Fi, and Drama
- Recent movies tend to perform better

## Business Recommendation
Streaming platforms should prioritize acquiring high-rated blockbuster movies in popular genres to maximize engagement and viewing time.
